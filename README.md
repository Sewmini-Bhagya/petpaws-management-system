# PetPaws Veterinary Hospital Management System

PetPaws is a web-based veterinary hospital management system designed to streamline and automate clinic operations, including appointment scheduling, pet management, medical records, and inventory tracking.

---

## Tech Stack

### Frontend

* React.js
* Tailwind CSS

### Backend

* Node.js
* Express.js

### Database

* MySQL

---

## Features

* JWT-based authentication
* Role-based access control (Admin, Vet, Receptionist, Client)
* Pet profile and medical record management
* Online appointment booking and scheduling
* Real-time queue tracking system
* Prescription and inventory management
* Notification system (email-based)
* Payment and invoice management
* Feedback and loyalty system

---

## Project Goal

The goal of PetPaws is to reduce manual workload in veterinary clinics, improve efficiency, and enhance client experience through automation and real-time system features.
